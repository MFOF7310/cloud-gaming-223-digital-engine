## 2. What Data We Collect

### 2.1 Discord Identifiers

- **Discord User ID** — unique identifier for tracking XP, economy, settings

- **Discord Username** — displayed in leaderboards, profiles, welcome messages

- **Discord Avatar URL** — for profile cards and embeds

- **Server (Guild) ID** — for per-server configuration and data isolation

- **Channel IDs** — for configured channels (welcome, log, ticket, etc.)

- **Role IDs** — for auto-role and permission systems

### 2.2 Activity Data

- **XP and Level** — progress in the leveling system

- **Credits and Economy** — virtual currency balances and transactions

- **Command Usage** — which commands are used (for statistics)

- **Vote Records** — Top.gg vote tracking with timestamps

- **Ticket History** — support ticket transcripts (stored per-server)

- **Security Logs** — threat analysis for moderation purposes

### 2.3 Message Data (Limited)

- **Message Content** — temporarily processed for:

  - AI responses (Lydia) — NOT permanently stored

  - Auto-moderation scanning — flagged content logged, normal content discarded

  - XP tracking — message counts only, content not retained

- **Message Timestamps** — for cooldown tracking and rate limiting

### 2.4 Image Data

- **User Avatars** — cached URLs from Discord's CDN (not stored locally)

- **Uploaded Images** — only if explicitly used with AI analysis commands (not permanently stored)

## 3. How We Use Data

| Data | Purpose | Retention |

|------|---------|-----------|

| User IDs | Identify users across features | Until bot removal or data deletion request |

| XP/Levels | Track server engagement | Same as above |

| Economy Data | Virtual currency system | Same as above |

| Server Settings | Per-server configuration | Until bot removed from server |

| Security Logs | Threat detection and moderation | 90 days |

| Ticket Transcripts | Support reference | Configurable (default: 180 days) |

| AI Conversations | Context for Lydia AI | 24 hours (rolling window) |

| Command Logs | Debugging and usage stats | 30 days |

## 4. Data Storage & Security

- **Database:** SQLite (WAL mode) stored on Bot-Hosting.net servers

- **Encryption:** Not currently encrypted at rest (Discord data is non-sensitive by design)

- **Access:** Only the bot owner has direct database access

- **Backups:** Weekly automated backups with 4-week retention

## 5. Data Sharing

We do **NOT** sell, trade, or share your data with third parties for commercial purposes.

Data is shared only with:

- **Discord API** — required for bot functionality

- **OpenRouter / Google Gemini** — for AI responses (message content sent temporarily)

- **Top.gg** — for vote verification (user IDs only)

- **Telegram** — only if Telegram bridge is explicitly enabled by server owner
