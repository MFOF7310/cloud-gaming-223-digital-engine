## 🛠️ Setup
1. Create a `.env` file with your `DISCORD_TOKEN`, `GEMINI_API_KEY`, `TIKTOK_USERNAME`, and `OWNER_ID`.
2. Ensure you have Node.js and `npm` installed.
3. Run `npm install` followed by `node index.js`.
